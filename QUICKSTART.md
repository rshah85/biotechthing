# Quick Start Guide

## Installation (2 Minutes)

### 1. Open Terminal (Mac) or Command Prompt (Windows)

**Mac**: Press `Cmd + Space`, type "Terminal", press Enter
**Windows**: Press `Windows key`, type "cmd", press Enter

### 2. Navigate to the Project

```bash
cd /Users/rishishah/Desktop/BPA_project/biotech-tracker
```

### 3. Install Dependencies

```bash
npm install
```

Wait 1-2 minutes for installation to complete.

### 4. Start the App

```bash
npm run dev
```

### 5. Open Your Browser

Go to: **http://localhost:3000**

---

## First Steps

1. **View Watchlist** - See AGIO, CYTK, CORT, DNLI, TVTX with real-time prices
2. **Click Refresh** - Update all stock prices
3. **Scan a Stock** - Try typing "ARVN" in the Stock Scanner
4. **Calculate Position** - Enter a price in Position Calculator
5. **Browse PDUFA Calendar** - See all upcoming FDA dates

---

## Key Features

- **Watchlist**: Track your stocks with live prices
- **Stock Scanner**: Search any ticker
- **PDUFA Calendar**: All FDA approval dates
- **Position Calculator**: Perfect position sizing
- **Dark Mode**: Click sun/moon icon (top right)
